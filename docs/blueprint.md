# **App Name**: Aman's Portfolio

## Core Features:

- Landing Page: Landing page with name, title, summary, and clear call to action.
- Projects Showcase: Projects section displaying project titles, descriptions, tech stacks, and links.
- Skills Highlight: Skills section listing proficiencies with visual aids.
- About Me: About Me section with a brief professional bio and personal aspirations.
- Contact Form: Contact section with a form for Name, Email, and Message, plus external links and Calendly integration.
- Navigation: Sticky top navigation bar with links to sections and a logo/name.

## Style Guidelines:

- Primary color: A deep blue (#293B5F) evoking professionalism and trust. The choice of blue reflects Aman's reliability and expertise.
- Background color: A very light blue (#E5E7EB), almost white, provides a clean and modern backdrop that makes the content pop without distracting from it.
- Accent color: A vibrant orange (#D98936) will draw attention to calls to action, and creates visual interest in combination with the deep primary blue.
- Font pairing: 'Inter' (sans-serif) for headlines and body text.
- Clean and modern design with a mobile-first responsive layout.
- Smooth hover effects and transitions to enhance user experience.
